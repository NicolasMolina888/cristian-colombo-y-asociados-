# Cristian Colombo & Asociados

Sitio web del estudio jurídico especializado en Derecho Penal del Dr. Cristian Gustavo Colombo.

## Características

- **Archivo HTML autocontenido** - Sin dependencias externas ni build step
- **Diseño responsive** - Optimizado para mobile y desktop
- **SEO optimizado** - Estructura semántica y meta tags
- **Navegación SPA** - Sistema de páginas con hash routing
- **Accesibilidad** - Contraste adecuado y soporte para `prefers-reduced-motion`

## Despliegue en Vercel

1. Sube este repositorio a GitHub
2. Conecta tu cuenta de Vercel con GitHub
3. Importa el repositorio
4. Vercel detectará automáticamente que es un sitio estático
5. Deploy automático

## Estructura

- `index.html` - Sitio web completo
- `CLAUDE.md` - Instrucciones del cliente para desarrollo
- Imágenes (logos y fotos)

## Tecnologías

- HTML5 semántico
- CSS3 moderno (variables CSS, Grid, Flexbox)
- JavaScript vanilla para navegación SPA
- Google Fonts (Playfair Display + Lato)
